# CASA Tier 2 — OWASP ASVS L2 Control → Evidence Map

Companion to `casa-readiness.md`. This maps each OWASP **ASVS Level 2** chapter to the
control that satisfies it and the **code path that is the evidence**, so it can be pasted
into the assessor's Self-Assessment Questionnaire (SAQ). Verify file paths still resolve
before submitting (they were accurate as of the `casa-tier2-hardening` branch).

Scope of assessment: the Zerrow web app (TanStack Start on Cloudflare Workers) + its Supabase
Postgres backend. AI classification is delegated to the Lovable AI gateway (`src/lib/ai.server.ts`).
